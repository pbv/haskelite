Welcome to Haskelite

Haskelite is a step-by-step interpreter for a subset of Haskell inspired by Pythontutor and Lambda lessons.

Go ahead and type an expression below and hit Evaluate to run it step-by step. Or you may start by choosing an example: recursive factorial; sum of squares; list reverse; list insertion; right and left folds; infinite lists. tree insertion; expression evaluation.

Haskelite is open-source software; please visit the Github page.
